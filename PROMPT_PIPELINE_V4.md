# Xobi Replace v4 流程（依据草图梳理）

## 输入
- 参考图：提供构图/光影/色调/文字区域。
- 产品图：真实产品，默认先做商业级精修。
- 可选：自定义文案、平台/比例/质量参数。
- 精修参考：`精修参考/` 下的案例质感（默认内建指令引用）。

## 固定提示词（始终启用）
- 先精修再合成：去手/杂物/噪点/褶皱/污渍，修缝线与纹理，抗锯齿，补 AO 与接触影。
- 形态保真：不改结构与比例，补齐缺失部位，贴合参考地面/桌面，避免悬浮。
- 构图/光影继承参考：仅替换主体，其余背景/道具/光影/色调/景深 100% 贴合参考。
- 文案规则：有文案放参考文字区，字体/描边/阴影/颜色跟参考一致；无文案则移除全部文字/Logo/水印。
- 输出格式：直接可用的中文生图 Prompt，无标题/解释/Markdown。

## 机器人建议（自动生成）
- 基于参考图/产品图分析，动态给出构图、光影、色调、材质、文字布局 5 条建议，并在前端以建议卡片展示。
- 前端“机器人正在回复和生成建议…”占位提示，返回建议数组时自动生成建议气泡。

## 用户/参数上下文
- 文案/话术：优先使用用户文案，否则移除文字。
- 参数：质量(1K/2K/4K)、宽高比、平台、版式、风格、背景、语言等。

## 输出
- `version: v4` 标记；返回生成 Prompt、分析结果、可选生成图（data URI）。

## 端到端逻辑
1) 分析参考图/产品图 → 生成 v4 Prompt（含固定提示词 + 机器人建议 + 用户/参数）。
2) 生成前自动插入精修要求（参考精修案例），确保“先精修后合成”。
3) 前端聊天区显示机器人回复占位，并渲染建议列表；用户可点选建议或直接确认生图。
